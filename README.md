## Installation

In hubot project repo, run:

`npm install hubot-newrelic-alert-webhook --save`

Then add **hubot-newrelic-alert-webhook** to your `external-scripts.json`:

```json
[
  "hubot-newrelic-alert-webhook"
]
```

## Sample Interaction

```
user1>> hubot hello
hubot>> hello!
```
